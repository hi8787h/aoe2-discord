# aoe2-discord
aoe2機器人分數 當前用於查詢分數,世紀帝國2排位分數區域分類(ex.天梯分數:1251->13+)
目前需要安裝 python

!ping - 輸出pong！
!myid -輸出discord id
!score -輸出您的1V1分數
!score @某人 -輸出某個人的1V1分數
!link - 使用者綁定 AoE2 帳號 輸入!link "url(aoe2insights)"     <----------請跟使用者說明
!update_score -除了會自動更新aoe2分數外 ,還會更新使用者的段位身分組

---------------------
!adminlink @某個人   #管理者幫忙登記某個人 AoE2 帳號 輸入 !link @某個人 "url(aoe2insights)"
!admindel @某個人   #管理者幫忙刪除某個人 AoE2 帳號 輸入 !link @某個人 "url(aoe2insights)"
